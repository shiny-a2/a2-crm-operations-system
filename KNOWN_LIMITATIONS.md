# Known Limitations

- Samples do not include a full schema migration layer.
- Reporting snapshots are discussed but not fully implemented publicly.
- Provider retry policy is intentionally simplified.
- No real customer timeline data is included.
- Screenshots are deferred until a synthetic interface is prepared.

## Operational Edge Cases To Track

- Reassigning a customer while another operator is editing.
- Provider accepted message but delayed delivery status.
- Manager report filters over large date ranges.
- Permission drift after role changes.
