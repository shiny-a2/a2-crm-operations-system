# Roadmap

This repository documents a public-safe CRM/customer operations architecture. It will stay focused on architecture, samples, and operational reasoning.

## Near Term

- Add a small capability matrix for operator, manager, and admin roles.
- Add a sample reporting repository that reads from snapshots instead of live timelines.
- Document provider retry and failure-state handling.
- Add a short ADR for keeping CRM operations inside WordPress.

## Later

- Add sanitized operator inbox wireframes.
- Add test fixtures for assignment rules and provider result handling.
- Add CSV export safety notes.

## Not Planned

- Publishing customer data or message examples.
- Publishing real provider integration code.
- Publishing private workflow rules.
